# ADAMSI_FGM Student Contribution

This folder contains a student-contributed face-verification adaptation of the ADAMSI_FGM attack integrated into the shared transfer-attack pipeline.

- **College:** SRM Institute of Science and Technology

## Attack
- **Implementation name:** `ADAMSI_FGM`
- **Type:** Adaptive momentum and adaptive-step transfer attack

## Reference paper
- **Title:** *On the Convergence of an Adaptive Momentum Method for Adversarial Attacks*
- **Authors:** Yuxiao Long, Xuanyu Yi, Jinglun Xu, Yuan Wu, Xiaosen Wang
- **Venue:** AAAI 2024
- **Paper:** https://ojs.aaai.org/index.php/AAAI/article/view/27997

## Important note
This repository stores the student-integrated CNN face-verification adaptation. The implementation is a practical adaptation of the paper’s adaptive-momentum idea inside the shared face-verification pipeline and should not be treated as an official reproduction of the original authors’ full implementation details.
